## ECOSYSTEM PROJECTS

DiaLabs designs, develops, and deploys scalable, production-grade applications with a focus on local-first intelligence, high-performance pipelines, and seamless user experiences.

### 01 // [PATENT_IQ.EVAL] - PatentIQ
* **Endpoint:** [patentiq.dialabs.tech](https://patentiq.dialabs.tech)
* **Repository:** [github.com/dialabs/patentiq](https://github.com/dialabs/patentiq)
* **Overview:** A serverless patent analytics platform. By marrying a high-concurrency Cloudflare Workers API backend with a dedicated Express LLM microservice, it parses patent documents, validates citations against SerpAPI, and computes multi-dimensional patentability scores in real-time.
* **Key Stack:** Next.js, Cloudflare Workers, Express, Gemini LLM, SerpAPI, pdfMake.

### 02 // [DIA_CHAT.SECURE] - Dia Chat
* **Endpoint:** [chat.dialabs.tech](https://chat.dialabs.tech)
* **Repository:** [github.com/dialabs/diachat](https://github.com/dialabs/diachat)
* **Overview:** A high-speed, local-first chat client prioritizing end-to-end user privacy. Built with secure real-time voice-to-text engines, customizable user presets, and state synchronization across sessions.
* **Key Stack:** React, Web Speech API, Tailwind CSS, LocalStorage Session State, Firebase.

### 03 // [DIA_MODERATOR.AUTOMATE] - Dia Moderator
* **Endpoint:** [mod.dialabs.tech](https://mod.dialabs.tech)
* **Repository:** [github.com/dialabs/diamoderator](https://github.com/dialabs/diamoderator)
* **Overview:** Scalable and automated safety suites designed to monitor, filter, and secure large-scale communities across Discord, Telegram, and other chat networks, ensuring 24/7 autonomous safety.
* **Key Stack:** Node.js, Express, Socket.io, Gemini AI, Discord.js, Telegram Bot API.
